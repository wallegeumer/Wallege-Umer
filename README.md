## Contact Form Setup

The contact form uses **Web3Forms**.

Form endpoint:

```html
https://api.web3forms.com/submit
```

The form includes a hidden Web3Forms access key:

```html
<input type="hidden" name="access_key" value="YOUR_ACCESS_KEY" />
```

When a visitor submits the form, the message is sent directly through Web3Forms.

## Deployment

This project is deployed on **Vercel**.

To deploy:

1. Push the project to GitHub.
2. Import the repository into Vercel.
3. Keep the main file as `index.html`.
4. Wait for deployment to complete.
5. Open the live Vercel URL.

Live URL:

```txt
https://wallege-umer-dev.vercel.app/
```
